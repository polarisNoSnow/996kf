本插件采用 vscode及其插件Chrome Extension Developer Tools开发
插件功能
对于收到console.kf.996sdk.net/currentChat网站用户信息及时播放警示音

## Features

- 暂无

## Install
### 1.下载Releases的version0.1并解压
### 2.打开chrome的扩展程序页面，将文件夹拖入即可
### 3.在指定网站打开一次插件的页面并点击保存设置（授权声音播放）
